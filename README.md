The goal of the company was to determine how many units of each product should be produced in order to maximize total profit without exceeding available resources. This type of problem is very common in manufacturing and production planning.

Next, the problem was converted into a mathematical model. Decision variables were defined for each product, representing the number of units to be produced. An objective function was created to maximize total profit. Multiple constraints were added to ensure that machine hours, labor hours, and raw materials were not exceeded. Additional upper-bound constraints were also included to represent market demand limits.
